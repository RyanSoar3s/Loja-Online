Loja Online Fictícia

Esta é uma loja online fictícia desenvolvida exclusivamente em Angular. O objetivo desse projeto foi criar uma interface interativa e responsiva, exibindo produtos armazenados em um arquivo JSON que atua como uma API. O projeto demonstra conceitos fundamentais presentes no framework, como componentização, rotas e integração com o serviço HttpClient.
